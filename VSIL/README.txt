This directory contains all the files necessary to run ionic liquid
simulations using GROMACS 5.0.7 with the OPLS-VSIL force field.

See paper:
Doherty, B.; Zhong, X.; Acevedo, O. "A Virtual Site OPLS Force Field for Imidazolium-Based Ionic Liquids" 
J. Phys. Chem. B, 2018, 122, 2962-2974. doi:10.1021/acs.jpcb.7b11996

NOTE: Do NOT mix OPLS-2009IL and OPLS-VSIL parameter files.
